Submission Workflow
  1. Fill out event details below
  2. Click "Submit for Approval" to send for bishopric review, or "Save as Draft" to save without notifying
  3. Bishopric will review and provide feedback
  4. Events appear on the calendar once approved and published

* Required field

Title*
Displayed publicly on the calendar and announcements page. e.g. Ward Conference, Ward BBQ Night
Type*
Calendar Event appears on calendar with date/time. Announcement shows as a notice without specific timing.
Organization*
Displayed as a public badge on the event listing.
Visibility
Turn off to restrict this event/announcement to logged-in users only.
Description
Shown publicly when viewers expand the event details. Supports basic HTML formatting.

Date*
Select the date this event occurs. Past dates are not allowed.
Check this if the event has no specific start/end time (e.g., all-day activity)
Start time*
What time does the event begin? Use 15-minute increments.
End time*
What time does the event conclude?

Location
Displayed publicly on the event listing. e.g. Cultural Hall, Stake Center

Attachments
Publicly visible in the event details. Max 5MB per file. Supports: images, PDFs
All submissions require bishopric approval before they appear on the public calendar.
